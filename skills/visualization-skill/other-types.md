# 可视化 Skill · 其他类型

适用：非 OCR 的纯文本、表格、列表等 API 类型。

---

## 左侧：原始 JSON

- 正常显示完整 JSON，提供一键复制、一键下载，见 [reference.md](reference.md)。

---

## 右侧：可视化框

- 仅保留一个空白框（如 `vis-card`），供用户根据业务自行设计可视化内容。
- 不做具体展示约定。

---

## 新增类型

若新 API 不属于上述类型，仍按左右分栏：左侧 JSON，右侧留空白框供用户设计。
