# 可视化 Skill · OCR / 作文识别类

适用：返回结构化文本、坐标或标注信息的 OCR 类接口。大部分业务场景属于此类。

---

## 左侧：原始 JSON

- **展示**：正常显示完整 JSON，可用 `pre` 配合 `JSON.stringify(data, null, 2)` 格式化。
- **操作**：提供一键复制、一键下载，见 [reference.md](reference.md)。

---

## 右侧：基于输入图的可视化

按以下三步实现：

1. **理解输出结构**：阅读技术文档或 JSON 文档，弄清输出包含哪些信息（如 bbox 坐标、题目区域、作答区、批改结果、切题框等）。
2. **在输入图上可视化**：将 JSON 中的坐标、标注等信息叠加到**用户上传的输入图片**上（框、高亮、标签等），使识别结果与原图一一对应。
3. **放入右侧**：将可视化后的图片（或原图 + overlay）放置在右侧 `vis-card` 内，可配下载标注图按钮。

实现细节见 [reference.md](reference.md) § 原图标注。
